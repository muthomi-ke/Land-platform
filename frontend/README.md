## LandPortal Frontend

React + Vite + Tailwind CSS landing page for a modern land selling platform (Airbnb-style for land).

### Tech stack

- **Build tool**: Vite (React + TypeScript)
- **Styling**: Tailwind CSS
- **Icons**: `lucide-react`
- **Animation**: `framer-motion`

### Getting started

1. **Install Node.js** (if you don&apos;t have it yet)
   - Recommended: Node 18+ from the official website (`https://nodejs.org`).

2. **Install dependencies**

   ```bash
   cd frontend
   npm install
   ```

3. **Run the dev server**

   ```bash
   npm run dev
   ```

   Then open the printed local URL (usually `http://localhost:5173`) in your browser.

### Customizing

- **Branding**: Update colors and fonts in `tailwind.config.cjs`.
- **Hero content**: Edit copy and search filters in `src/App.tsx` (`Hero` and `SearchCard` components).
- **Plots**: Adjust or replace demo data in the `featuredPlots` array in `src/App.tsx`.
